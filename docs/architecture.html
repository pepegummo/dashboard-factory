3 · API Reference
Method Route Handler Notes
GET /api/health inline {"status":"ok"}
GET /api/factories FactoryHandler.List returns all factories
GET /api/factories/{id}/machines FactoryHandler.ListMachines machines belonging to factory
GET /api/metrics TemplateHandler.Metrics fixed catalog: 6 metrics with key/label/unit/min/max
GET /api/templates TemplateHandler.List
POST /api/templates TemplateHandler.Create validates widget types (gauge|line|kpi|status|table) + sizes
GET /api/templates/{id} TemplateHandler.Get includes parsed widgets array
PUT /api/templates/{id} TemplateHandler.Update
DELETE /api/templates/{id} TemplateHandler.Delete 409 if template is used by a dashboard
GET /api/dashboards DashboardHandler.List
POST /api/dashboards DashboardHandler.Create validates template+factory exist; all machines ∈ factory; inserts in tx
GET /api/dashboards/{id} DashboardHandler.Get includes template, factory, pages with machine objects
DELETE /api/dashboards/{id} DashboardHandler.Delete
GET /api/machines/{id}/telemetry TelemetryHandler.Get no DB read; sine-wave simulation seeded by fnv32a(machineID:metricKey)
